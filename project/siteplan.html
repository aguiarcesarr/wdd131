Shadowhunters Universe Guide

Site Name

Shadowhunters Universe Guide

This name represents a fan-made guide to Cassandra Clare’s Shadowhunters book series. It reflects the site's goal of helping fans explore the complex universe by organizing timelines, character bios, and interactive elements in one accessible place.

Site Purpose

The purpose of this site is to provide a comprehensive and user-friendly guide to the Shadowhunters universe. It aims to:

  • Organize timelines of events across the series
  • Offer detailed character biographies
  • Include interactive elements like quizzes and polls
  • Serve as a community hub for fans to share theories and discussions

User Scenarios

  • “I just started reading the Shadowhunter books—what's the best reading order or timeline to follow?”
  • “Who is Tessa Gray and how is she connected to other main characters across the series?”

Color Schema

This project uses a dark academia aesthetic inspired by the Shadowhunters universe. Below are the primary colors and their usage:

  • --bg-color: #1b1b1b – Used as the main background color for a moody, immersive atmosphere.
  • --text-color: #f5f5f5 – Main text color for high readability on dark backgrounds.
  • --accent-color: #9d4edd – Used for section titles, links, and highlights.
  • --highlight-color: #5a189a – Used for visual accents like borders and focus outlines.
  • --heading-color: #e0aaff – Used for the main page title to give it a glowing look.
